## LayerSwipe-jsapi4.x
基于ArcGIS JS API 的地图卷帘插件

### 基于作者GIS-RC的[LayerSwipe-jsapi4.10](https://github.com/GIS-RC/LayerSwipe-jsapi4.10)代码重构
  1.重构插件的declare方式；  
  2.添加支持MapView；  
  3.优化鼠标移动监听，优化left和top计算；
